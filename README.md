## Day2    3.5
> 中正纪念堂 -> 士林官邸 -> 台北故宫博物院 -> 宁夏夜市
### 中正纪念堂（约1~3小时）
* 主要项目
1. 参观
2. 在第4层整点时间会有守卫兵交接仪式观看。
3. 升旗仪式6:30 | 降旗仪式17:10
* 交通
搭乘淡淡水信义线 在中正纪念堂站下车然后步行即可到达
* 门票：免费
* 开放时间
09:00-18:00 (1月1日-12月31日 周一-周日)

### 士林官邸（1~3小时）
* 主要项目
1. 参观
* 交通
乘坐捷运淡水线、信义线至士林站，2号出口，左转过中山北路即可到达
* 门票
100 TWD ≈  22 RMB
* 开放时间
09:00-12:00,13:30-17:00 (1月1日-12月31日 周二-周日)

### 台北故宫博物院（约3小时）
* 主要项目
1. 参观
+ 交通
1. 搭乘捷运淡水信义线  士林站下车，由1号出口出站，往中正路转乘公交车
红30至故宫博物院本院正馆负一楼门口下车
255、304、815、小18、小19至故宫博物院本院山脚广场前下车。
2. 搭乘捷运文湖线（棕色1号线）
至剑南路站下车，转乘公交车棕20往故宫博物院本院正馆负一楼门口下车
至大直站下车，由3号出口出站，右转直行约百米，转乘公交车棕13至故宫博物院本院山脚广场前下车
+ 门票
350 TWD ≈ 76 RMB
+ 开放时间
08:30-18:30 (1月1日-12月31日 周一-周五)
08:30-21:00 (1月1日-12月31日 周六-周日)
+ TIPS
1. 讲解器 100TWD，有可能需要压通行证
2. 不能带双肩包，寄存柜在负一楼（10元硬币）
3. 全程蹭义工讲解…

### 宁夏夜市
+ 交通
乘捷运淡水线或北投线至双连站下，再步行约10分钟。
+ 开放时间：18:00-24:00
